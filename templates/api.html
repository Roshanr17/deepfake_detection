API Documentation

Note: The API is currently in open beta. Please report any issues or feature requests.

Getting Started

The Deepfake Detector API allows you to programmatically analyze videos for deepfake content. All requests are processed asynchronously.

Base URL

https://api.deepfakedetector.io/v1

Authentication

Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY
Get your API key: Sign up and generate an API key from your dashboard.

Endpoints

POST
/analyze
Upload a video for deepfake analysis

Request Parameters

Parameter Type Description
file file Video file (MP4, MOV, AVI, WebM, MKV)
async boolean Optional. Return immediately with task ID (default: false)

Successful Response

200 OK
{ "status": "success", "prediction": "fake", "confidence": 0.92, "real_probability": 0.08, "fake_probability": 0.92, "processing_time": 45 }

Error Response

400 Bad Request
{ "status": "error", "message": "No valid video file provided" }
GET
/status/{task_id}
Check the status of an async analysis task

Request Parameters

Parameter Type Description
task_id string Task ID from the analyze request

Response

200 OK
{ "task_id": "abc123xyz", "status": "completed", "result": { ... } }

Code Examples

Python

import requests # Upload a video for analysis headers = {'Authorization': 'Bearer YOUR_API_KEY'} files = {'file': open('video.mp4', 'rb')} response = requests.post( 'https://api.deepfakedetector.io/v1/analyze', headers=headers, files=files ) result = response.json() print(f"Prediction: {result['prediction']}") print(f"Confidence: {result['confidence']*100:.1f}%")

JavaScript/Node.js

const axios = require('axios'); const FormData = require('form-data'); const fs = require('fs'); const form = new FormData(); form.append('file', fs.createReadStream('video.mp4')); axios.post( 'https://api.deepfakedetector.io/v1/analyze', form, { headers: { ...form.getHeaders(), 'Authorization': 'Bearer YOUR_API_KEY' } } ).then(response => { console.log(response.data); });

cURL

curl -X POST https://api.deepfakedetector.io/v1/analyze \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "file=@video.mp4"

Rate Limits & Quotas

  • Free Tier: 10 requests per hour, 100 MB per request
  • Pro Tier: 100 requests per hour, 500 MB per request
  • Enterprise: Custom limits, dedicated support

Error Codes

Code Message Description
400 Bad Request Invalid request format or missing required parameters
401 Unauthorized Invalid or missing API key
413 Payload Too Large File size exceeds limit for your tier
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error during processing
